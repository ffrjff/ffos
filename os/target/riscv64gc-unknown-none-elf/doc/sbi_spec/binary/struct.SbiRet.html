Expand description

SBI functions return type

SBI functions must return a pair of values in a0 and a1, with a0 returning an error code. This is analogous to returning the C structure SbiRet.

Note: if this structure is used in function return on conventional Rust code, it would not require to pin memory representation as extern C. The repr(C) is set in case that some users want to use this structure in FFI code.
